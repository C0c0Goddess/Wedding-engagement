# Loyalty Ova Love Wedding System V3 Complete

This is the gameplay wedding script. In this bundle it is paired with the separate Aura map resource.

## Includes
- Wedding planner UI with auto-popup fix
- Proposal system
- Accept/decline popup
- Ring shop
- Marriage/divorce/status
- Marriage licenses
- Mr./Mrs. nametags
- Venue booking, including Aura Wedding Venue entry
- Guest invitations
- Shared couple account
- Admin command
- Discord logging support
- SQL install

## Commands
/weddingplanner
/propose [player id]
/marry [player id]
/divorce
/marriagestatus
/marriagelicense [player id]
/changemarriedname [lastname]
/startceremony [player id]
/coupleaccount
/weddingadmin
